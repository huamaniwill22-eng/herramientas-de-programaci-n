# Calculadora GUI con Tkinter

Una calculadora gráfica sencilla y funcional desarrollada en Python utilizando la librería **Tkinter**.

## ✨ Características
* Interfaz gráfica intuitiva y responsiva.
* Operaciones matemáticas básicas (Suma, Resta, Multiplicación y División).
* Manejo de errores (evita el cierre del programa al dividir por cero o ingresar expresiones inválidas).

## 🚀 Requisitos y Ejecución
Solo necesitas tener Python instalado (Tkinter viene incluido por defecto).
